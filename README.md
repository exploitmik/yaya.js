# yaya.js
A lightweight lib for light animations

Add attribute "data-yaya" in element for anime. Available types:

* yayaClipToRight
* yayaClipToLeft
* yayaClipCircle
* yayaToUp
* yayaToDown
* yayaToRight
* yayaToLeft
* yayaBlurOut

Other attributes availables:
* data-yaya-duration
* data-yaya-delay
